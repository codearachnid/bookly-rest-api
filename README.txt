## Introduction

Bookly REST API exposes the Bookly resources to the public making it easier to extend the functionality or serve it to other systems.

## Installation

 * Make sure the `Bookly` plugin is installed and activated.
 * Make sure the `Bookly-Pro` plugin is installed and activated.
 * Download, install & activate Bookly REST API plugin.

## Usage

* You should see a submenu `Bookly REST API`, under the `Bookly` menu.
* Go to the Bookly REST API submenu, click on the endpoint you want to use, and press `Try it out`.
* Enter the parameters (if required) and then press `Execute`.
* You should see the response of the the endpoint executed.

## Troubleshoot

If you get your site HTML content, make sure you have the permalinks using urlrewrite. To do so go to Settings->Permalinks and choose for instance the `Post name` option.
